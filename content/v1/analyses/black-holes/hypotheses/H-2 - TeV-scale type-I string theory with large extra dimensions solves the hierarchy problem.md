String embedding of the ADD large-extra-dimensions proposal (companion to hep-ph/9803315). The observed weakness of gravity at long distances is attributed to the large volume of the extra dimensions in which only closed-string gravity propagates, while SM matter is automatically localised on a D3-brane. The scenario requires an O(1) string coupling and new dimensions much larger than the weak scale.
